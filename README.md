## 🎯 Por Que Este Curso Existe?

Estamos vivendo a **maior revolução tecnológica desde a internet**: a **Era dos Agentes de IA**.

Não estamos falando de chatbots que respondem perguntas. Estamos falando de **sistemas inteligentes que agem, decidem e executam** - automatizando tarefas complexas que antes exigiam equipes inteiras.

**A questão não é SE sua empresa/carreira será impactada pela IA. É QUANDO - e se você estará preparado.**

## 🚀 O Que Você Vai Aprender

### 📚 **8 Módulos Progressivos** (60 horas de conteúdo)

<details>
<summary><b>Módulo 1: Fundamentos de IA Generativa</b> (8h) - 🟢 Iniciante</summary>

**O que você vai dominar:**
- Como LLMs realmente funcionam (sem a mágica)
- Arquitetura Transformer explicada visualmente
- Tokenização, Embeddings e Multimodalidade
- Parâmetros de geração (Temperature, Top-p, Top-k)

**Projeto prático:** Gerador de Ideias Multimodal

**Versões disponíveis:**
- 🟢 Conceitual (para não-técnicos)
- 🟡 Prático (com ferramentas no-code)
- 🔴 Técnico (código completo em Python)

</details>

<details>
<summary><b>Módulo 2: Engenharia de Prompts Avançada</b> (6h) - 🟢 Iniciante-Intermediário</summary>

**O que você vai dominar:**
- Anatomia de um prompt perfeito (PCFT framework)
- Zero-Shot vs Few-Shot Prompting
- Chain of Thought (CoT) e Self-Consistency
- Tree of Thoughts para problemas complexos
- System Prompts: a "constituição" de um agente

**Projeto prático:** Sistema de Análise de Sentimento com CoT

</details>

<details>
<summary><b>Módulo 3: Frameworks - LangChain</b> (10h) - 🟡 Intermediário</summary>

**O que você vai dominar:**
- Arquitetura modular do LangChain
- Chains: conectando blocos de lógica
- RAG (Retrieval-Augmented Generation)
- Agentes autônomos com ReAct
- Integração com 50+ Vector Stores e APIs

**Projeto prático:** ChatPDF - Converse com seus documentos

</details>

<details>
<summary><b>Módulo 4: Agentes Autônomos com Agno</b> (5h) - 🟡 Intermediário</summary>

**O que você vai dominar:**
- Filosofia minimalista de desenvolvimento
- Criação rápida de agentes especializados
- RAG e memória persistente simplificados
- Sistemas multi-agentes elegantes

**Projeto prático:** Assistente Pessoal de Tarefas

</details>

<details>
<summary><b>Módulo 5: Sistemas Multi-Agentes com CrewAI</b> (12h) - 🔴 Avançado</summary>

**O que você vai dominar:**
- Orquestração de equipes de IA especializadas
- Design de agentes com papéis definidos
- Processos Sequenciais vs Hierárquicos
- Observabilidade com AgentOps
- Casos de uso corporativos reais

**Projeto prático:** Agência de Criação de Conteúdo Automatizada

</details>

<details>
<summary><b>Módulo 6: Model Context Protocol (MCP)</b> (8h) - 🔴 Avançado</summary>

**O que você vai dominar:**
- Padronização de ferramentas para IA
- Criação de servidores MCP
- Integração universal de APIs e sistemas
- Interoperabilidade entre plataformas

**Projeto prático:** Servidor MCP para Gerenciamento de Projetos

</details>

<details>
<summary><b>Módulo 7: Deploy e Produção</b> (6h) - 🔴 Avançado</summary>

**O que você vai dominar:**
- Arquiteturas de deploy escaláveis
- FastAPI para exposição de agentes
- Containerização com Docker
- Deploy em AWS, GCP, Azure
- Monitoramento e otimização de custos

**Projeto prático:** Deploy da Agência na Nuvem

</details>

<details>
<summary><b>Módulo 8: Ferramentas e Ecossistema</b> (5h) - 🟡 Intermediário</summary>

**O que você vai dominar:**
- Hugging Face Hub completo
- Modelos open-source (Llama, Mistral, etc)
- Ollama para execução local
- Fine-tuning e otimização
- Ferramentas de observabilidade

**Projeto prático:** Ambiente de Desenvolvimento IA Completo

</details>
