## 📞 ATALHOS ÚTEIS

### Comandos Git Úteis

```bash
# Ver status dos arquivos

git diff

# Adicionar mudanças
git add .

# Fazer commit
git commit -m "descrição"

# Ver histórico
git log --oneline -10

# Ver branches

```

### Comandos do Sistema

```bash
# Contar linhas de um arquivo
wc -l arquivo.md

# Ver tamanho dos arquivos
```
